# Proyecto de Bases de Datos - Liga Profesional de Fútbol

---

## Descripción del Proyecto
Este proyecto forma parte de la Escuela Politécnica Nacional (EPN), en la Escuela de Formación de Tecnólogos (ESFOT), 
para la asignatura de Base de Datos. Consiste en el diseño e implementación de un sistema de gestión de bases de datos 
relacional para organizar y administrar información de equipos, jugadores, entrenadores, árbitros, estadios, torneos y partidos.
